# Appearance

Workspace Hub includes a persisted appearance control in the app header.

## What it does

- lets you switch between built-in visual presets
- supports light and dark mode for each preset
- saves the current selection locally in the browser

## Current presets

- `Preset 1 · Ops`
  GitHub and DigitalOcean-style dashboard density using Dracula Classic and Alucard palette choices.
- `Preset 2 · GitHub`
  A flatter GitHub-style layout based on the local Dracula Pro `github-master` reference.
- `Preset 3 · Studio`
  A softer Dracula Pro IDE-style variant with stronger purple and pink component accents.
- `Preset 4 · Control`
  A darker control-room dashboard variant with brighter cyan, purple, and pink accents.
- `Preset 5 · Workspace`
  The original warm workspace styling kept as the baseline option.

## Notes

- preset numbering is user-facing and follows the current dropdown order
- changing the preset does not affect repo data or runtime behaviour
- appearance settings are local to the current browser profile
